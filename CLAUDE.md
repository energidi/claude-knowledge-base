# Project Overview
This repository stores my Claude conversation history and project rules.

## Standard Procedures
- **Saving Chats:** Export Claude chats as Markdown and save them in `docs/chats/`.
- **Naming Convention:** Use `YYYY-MM-DD-topic-name.md`.
- **Rules:** Specific behavior instructions are stored in `.claude/rules/`.

## Tone and Style
- Be concise and direct.
- Use technical language where appropriate.
- Avoid unnecessary conversational filler.
